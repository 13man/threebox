# 诗词Live - 快速开始指南

## 🎯 功能位置

「诗词Live」功能位于应用主页底部导航栏的第4个Tab:

```
匣(📜) → 著名诗人(👤) → 趣味填词(📝) → 诗词Live(🎬) → 我的(👤)
```

点击 **🎬 诗词Live** 即可进入。

## 🎮 操作说明

### 1. 基础操作

#### 📱 滑动屏幕
- **操作**: 在画布上滑动手指
- **效果**: 月亮跟随手指移动
- **范围**: 横向 10%-90%, 纵向 10%-60%

#### ▶️ 播放朗诵
- **操作**: 点击中央绿色播放按钮
- **效果**: 
  - 开始逐字朗诵诗词
  - 当前朗诵的字会高亮显示
  - 配合动画营造沉浸式体验

#### ⏸️ 暂停朗诵
- **操作**: 朗诵时点击暂停按钮
- **效果**: 暂停朗诵,高亮停留在当前字

#### ◀️ ▶️ 切换诗词
- **左箭头**: 上一首诗词
- **右箭头**: 下一首诗词
- **效果**: 
  - 自动切换场景和粒子效果
  - 重置月亮位置
  - 停止当前朗诵

### 2. 交互细节

#### 月亮移动
- **拖动流畅度**: 实时跟随,无延迟
- **边界限制**: 不会移出画面
- **光晕效果**: 月亮周围自动生成柔和光晕

#### 粒子动画
- **自动适配**: 根据诗词类型自动选择粒子
- **持续飘落**: 粒子循环生成,永不停歇
- **边界复用**: 飘出画面的粒子会在顶部重生

#### 诗句高亮
- **逐字高亮**: 朗诵进度同步高亮
- **颜色变化**: 高亮字变为绿色加粗
- **动画过渡**: 200ms平滑过渡

## 🔧 常见问题

### Q1: 画面不流畅?
**A**: 
- 关闭其他后台应用
- 确保设备性能良好
- 重启应用重新加载

### Q2: 朗诵无声音?
**A**:
- 检查设备音量
- 确认系统未静音
- 检查权限设置

### Q3: 月亮移动不跟手?
**A**:
- 确认在画布区域滑动
- 避免快速滑动
- 重启页面重试

### Q4: 粒子效果不显示?
**A**:
- 等待页面完全加载
- 检查Canvas渲染是否正常
- 切换到其他诗词再切回
